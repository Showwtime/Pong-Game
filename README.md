# Pong Game

A basic pong game using SVGs.

## Setup

**Install dependencies:**

`> npm i`

**Run locally with Webpack Dev Server:**

`> npm start`

**Build for production:**

`> npm run build`

## Keys

**Player 1:**
* a: up
* z: down
* x: right
* shift: left

**Player 2:**
* ▲: up
* ▼: down
* left-arrow: left
* right-arrow: right
